Blocks orb/ring jumps in Geometry Dash by overriding the reward system. The mod hooks into GameLevelManager to return 0 rewards for all levels, effectively disabling orb progression.
class $modify(NoOrbs, GameLevelManager) {
  int getRewardForLevel(...) {
    // Bloquea completamente las orbes
    return 0;
  }
};
gameplay bugfix Windows Android Mac iOS
This is a C++ game mod for Geometry Dash using the Geode SDK. To build it, you need the Geode SDK installed and CMake 3.21+. Set the GEODE_SDK environment variable and run cmake -B build && cmake --build build.
